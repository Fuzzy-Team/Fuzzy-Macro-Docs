# Private Server Link

{% stepper %}
{% step %}
### Copy your VIP server link

Open Bee Swarm Simulator -> Servers -> click the three dots on your server -> Configure -> scroll down and copy the VIP server link.

![VIP server link screenshot](<../.gitbook/assets/image (27)>)
{% endstep %}

{% step %}
### Convert the link

Paste the copied link into a new browser tab and load the page. After the page loads, you should see the converted format, for example:

https://www.roblox.com/games/123123123/Bee-Swarm-Simulator?privateServerLinkCode=examplenumbers
{% endstep %}

{% step %}
### Use the converted link

Take the converted link shown in the address bar and paste it into the macro â€” it will reconnect using that link.
{% endstep %}
{% endstepper %}

{% hint style="info" %}
If you don't have a server, you can join one such as the official Bee Swarm Simulator server or other public servers. Just make sure the server you use provides the correct VIP/private server link format.
{% endhint %}
