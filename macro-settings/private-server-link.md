# Private Server Link

{% stepper %}
{% step %}
### Copy your VIP server link

Open Bee Swarm Simulator -> Servers -> click the three dots on your server -> Configure -> scroll down and copy the VIP server link.

![VIP server link screenshot](<../.gitbook/assets/image (27)>)
{% endstep %}

{% step %}
### Open up macro

Go to the config tab and paste in the url.
{% endstep %}
{% endstepper %}

{% hint style="info" %}
If you don't have a server, you can join one such as the official Bee Swarm Simulator server or other public servers.
{% endhint %}
