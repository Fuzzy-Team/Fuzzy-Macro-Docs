# Keyboard

Make sure that you have the ABC input source added and selected.

To add it, click the "+" button → English → ABC.

How to switch input sources:

* Enable the "Show input menu in menu bar" checkbox.
* From your menu bar, click the flag/language icon and switch to ABC.
