# Display Settings

## Display Resolutions

Your screen resolution should be either 2880x1800 or 1920x1080 for optimal performance. If your screen resolution isn't one of these, it can mess up certain features like quest detection.

## Color settings

Set your color profile to "sRGB IEC61966-2.1"\
Alternatively, set your preset to "Internet & Web (sRGB)"
