# Disable Roblox scaling

{% hint style="warning" %}
**This step is not applicable for all Macs. If you do not see the option described below, you can skip this step. It mainly applies to Macs with built-in (embedded) cameras.**
{% endhint %}

{% stepper %}
{% step %}
### Enable Roblox Scaling (if available)

Close Roblox before making changes.
{% endstep %}

{% step %}
### Open Finder

Go to Applications, right‑click Roblox, and choose "Get Info".
{% endstep %}

{% step %}
### Enable scaling

In the Get Info window, enable "Scale to fit below built‑in camera" if the option is present.
{% endstep %}

{% step %}
### If the option is not present

If you can't find this option, you can ignore this step and proceed—it's optional.
{% endstep %}

{% step %}
### Re-open Roblox

Re-open Roblox. If you take a screenshot of the whole screen while in Roblox, there should not be any black bars around the screenshot image.
{% endstep %}
{% endstepper %}
