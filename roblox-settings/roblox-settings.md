# Roblox Settings

Make sure your Roblox settings are set as follows:

* Camera Mode: **Classic**
* Movement Mode: **Default (Keyboard)**
* Fullscreen: **On**
* Performance Stats: **Off**
* Micro Profiler: **Off**
* Camera Inverted: **Off**
* UI Navigation Toggle: **On**
* Experience Language: **English**
* Background Transparency: **0**

{% hint style="info" %}
NOTE: If you are using fast flags, ensure that "DFIntDebugFRMQualityLevelOverride" is set to 4 or above. (If you don't know what that is you can ignore it)
{% endhint %}
